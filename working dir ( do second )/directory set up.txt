now that we have done the " Server set Up " we are going to create our directory that our service will be in when using docker services!!!

since we are working with the root user : mkdir /root/Docker

Under Docker folder we are going to be placing our docker services with there .yaml files. ( emby , homer , kasm ect ....)

(For the music website we are not going to be using docker.)

We are going to store our nginx configs for ssl certs with revers proxy in /etc/nginx/sites-available

We have two sites : one public under 443 ( music site ) and one private under 8443 ( for our servises ( emby etc ....))

The static files for the music site will be under /var/www/sitename , and the .yaml files for our services under the dir we created in the first steps
