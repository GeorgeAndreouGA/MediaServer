Kasm workspace ( You have to install Docker Engine and Docker Compose )

It may not work for the first time.If it doesn't delete all the kasm files and try again.

diable your firewall and after the instalation turn it back on

1. curl -O https://kasm-static-content.s3.amazonaws.com/kasm_release_1.17.0.7f020d.tar.gz
2. tar -xf kasm_release_1.17.0.7f020d.tar.gz
3. sudo bash kasm_release/install.sh -L < choose the port that it will listen e.g 8443 >


Go to : https://<server-ip>:< port from step 3. > 

auto renew ssl certs : place the " kasm-selfsigned-renew.sh " under /root/Docker/kasm_release

go to crontab -e (root)  : and add =>    0 0 1 1 *  /root/Docker/kasm_release/kasm-selfsigned-renew.sh

it will exicute the .sh every year to auto regenerate the ssl certs
